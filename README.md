# IKEA Data Analysis

This project contains data analysis of IKEA products using Python and Jupyter Notebook.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Files
- code.ipynb — main notebook
- ikea.csv — dataset

## Project Goals
- explore IKEA product data
- visualize trends
- analyze prices
- apply machine learning
